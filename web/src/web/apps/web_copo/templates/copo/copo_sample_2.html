Samples - Getting started

The samples page provides a platform for describing and managing biological samples, obtained as part of a project.

The description functionality presents a wizard that guides the user through a step-by-step process of sample metadata attribution.

Use the icon to initiate sample description and follow the steps presented by the wizard.

Quick Tour: To explore the various controls available on this page, click the icon.

Sample Description - Getting started

To start descrbing your samples, click on the icon or, under the Inspect tab, click the 'Describe' button. The view should change to the Describe tab, with the description wizard initialised. Follow the sequence of steps presented by the wizard to describe your samples.

  1. Discard You can discard the current description, and exit the wizard, at any time by clicking the discard button.
  2. Info Use the info (information) button to access relevant messages in each stage of the description.
The 'Info' side pane will be employed to display relevant messages through different stages of a description.
Use the 'Help' side pane to access more information on describing samples.
